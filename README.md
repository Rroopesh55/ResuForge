# ResuForge

**Resume Format-Aware AI Optimization System**

A "Resume Operating System" that optimizes resumes for specific job descriptions while strictly adhering to the user's original layout and formatting constraints.

## Architecture

- **Frontend**: Next.js 14+ (App Router), TailwindCSS, Shadcn/UI
- **Backend**: FastAPI (Python 3.10+)
- **AI Engine**: Ollama (Local)
- **Database**: Supabase
